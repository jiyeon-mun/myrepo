함수 선언


    function 함수명([매개변수[, 매개변수[, ...]]]) {
        연산 코드;
        [return [결과값];]
    }

예제 코드


    function func1(x) {
        var result = x + 10;
        return result;
    }

익명 함수


    var f2 = function([매개변수[, 매개변수[, ...]]]) {
        연산 코드;
        [result [결과값];]
    }

    // 실행
    f2([전달인자[, 전달인자[, ...]]]);

예제 코드


    function func2() {
        var f2 = function() {
            return 10;
        }
        var result = f2();
        return result;
    }

호출 없이 바로 실행하는 함수


    var y = function([매개변수[, 매개변수[, ...]]]) {
        연산 코드;
        [return [결과값];]
    }([전달인자[, 전달인자[, ...]]]);

예제 코드


    function func3(x) {
        var y = function() {
            return 10;
        }();
        var result = x + y;
        return result;
    }

자바스크립트 함수의 매개변수 처리

자바스크립트의 함수는 전달인자의 수와 매개변수의 수가 동일하지 않아도 함수가 동작되도록 구성되어 있다.

매개변수의 수보다 전달인자의 수가 많은 경우에는 arguments 배열에 순차적으로 저장되며, 매개변수의 수보다 전달인자의 수가 적은 경우에는 매개변수에 전달되지 않은 인자에 대해 undefined 로 처리가 된다.

팁 : arguemtns 배열에는 모든 전달인자가 저장되어 있다.

예제 코드


    function func4(x, y) {
        console.log("x: " + x);
        console.log("y: " + y);
        console.log("arguments: ");
        for(var v of arguments) {
            console.log("\t\t\t" + v);
        }
    }

함수의 전달인자로 함수를 전달

함수의 전달인자로 함수를 전달할 때 '함수명'만 적어야 한다.

func5(func6) : func5 함수의 전달 인자로 func6 함수를 사용
func5(func6()) : func5 함수의 전달 인자로 func6 함수의 실행 결과를 사용

예제 코드


    function func5(x) {
        return x();
    }

    function func6() {
        return 10;
    }
