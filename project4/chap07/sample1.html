배열

var 변수명 = [값1, 값2, 값3, ...]
위와 같은 형식으로 사용

배열에 담을 수 있는 데이터 타입(자료형)은 제한이 없다.
서로 다른 자료형을 가진 값들을 하나의 배열에 저장할 수 있다.
자바에서의 리스트 컬렉션과 비슷하다.

new 연산자와 Array 객체를 통해 생성 가능

var 변수명 = new Array();
var 변수명 = new Array(크기); 크기를 지정해도 배열에 원소를 계속해서 저장할 수 있다.
var 변수명 = new Array(값1, 값2, ...);
